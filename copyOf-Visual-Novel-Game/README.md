# Visual Novel Game
By: Ethan Hom and Cristian Parker

## Project Overview
CSE 111: Database Systems project. The game implements a backend and frontend using Python and SQLite3.

    git add .

    git push


## Features

### Backend Operations (Game Designer)
- **Character Management**: Create, read, update, delete characters
- **Location Management**: Manage backgrounds and locations
- **Scene Management**: Create and organize scenes
- **Sprite Management**: Link character sprites with expressions
- **Dialogue System**: Add and edit dialogue lines
- **Choice System**: Create branching narrative paths
- **Event System**: Define and trigger game events

### Frontend Operations (Player)
- **Start New Game**: Initialize game from starting scene
- **Dialogue Display**: Show character dialogue with sprites
- **Scene Navigation**: Automatic progression through scenes
- **Choice Selection**: Player decision-making
- **Event Triggering**: Automatic event handling
- **Save/Load System**: Preserve game progress
